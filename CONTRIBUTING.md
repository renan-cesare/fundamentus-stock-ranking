# Contributing

Thank you for your interest in this repository.

This project is provided primarily as a **portfolio and study reference** for web scraping and data analysis using Python, Selenium and Pandas.

## Scope

Contributions are welcome as long as they remain aligned with the educational purpose of the project:

- Documentation improvements
- Small refactors for readability
- Minor bug fixes
- Improvements related to stability and execution

This project is not intended to be a production-grade scraper.

## Guidelines

- Do not include private/internal URLs or sensitive data
- Avoid running scraping routines at high frequency
- Keep changes minimal and focused
- Avoid introducing unnecessary architectural complexity

## How to contribute

1. Fork the repository
2. Create a branch
3. Open a pull request describing what changed and why

By contributing, you agree that your contributions will be licensed under the MIT License.
