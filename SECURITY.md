# Security Policy

## Supported Versions

This repository is provided as a portfolio/study reference.
Security updates may be applied when relevant, but no formal support is guaranteed.

## Reporting a Vulnerability

If you identify a potential security issue, please avoid opening a public issue with sensitive details.

Instead, report it privately by contacting the repository owner.

## Usage notes

- This project scrapes a public website (Fundamentus). Avoid running it at high frequency.
- Do not add credentials, tokens, or private/internal URLs.
- This project is not intended for production environments without proper review and adaptations.
